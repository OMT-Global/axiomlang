Part of #236. Add Ed25519 asymmetric signing primitives.

## Scope

- Host intrinsics: `crypto_ed25519_keygen(): (Vec<u8>, Vec<u8>)` (returns `(public_key, secret_key)`), `crypto_ed25519_sign(secret_key, message): Vec<u8>`, `crypto_ed25519_verify(public_key, message, signature): bool`.
- `std/crypto_sign.ax` typed wrapper module gated on `crypto`.
- Document key formats (32-byte public, 64-byte secret per RFC 8032).

## Acceptance

- Pass fixture: keygen → sign → verify round-trip.
- Pass fixture: tampered signature fails verify.

## Out of scope

- X25519 key agreement, Ed448 — separate sub-issues.
